 Assignment 7 – JavaScript Problem Solving Description

This repository contains solutions to 10 JavaScript problem-solving tasks from the SCIC-12 course.
The problems focus on basic JavaScript concepts such as functions, loops, conditionals, arrays, and string methods.

## Problems Solved

1.Reverse a String – Reverses a given string.

2.Count Vowels in a String – Counts the number of vowels in a string.

3.Check for Palindrome – Checks whether a string reads the same backward.

4.Find Maximum Number – Finds the largest number in an array.

5.Remove Duplicates from an Array – Removes duplicate numbers.

6.Sum of All Numbers in an Array – Calculates total sum.

7.Find Even Numbers in an Array – Returns all even numbers.

8.Capitalize First Letter of Each Word – Capitalizes first letters of words.

9.ind Factorial of a Number – Calculates factorial using a loop.

10.PingPong Challenge – Prints numbers 1–20 with “Ping”, “Pong”, or “PingPong” rules.

## How to Run the Code

Clone the repository:

git clone :  https://github.com/rimiakter4/Problem-Solve.git


Open terminal in the folder.
Run each problem file with Node.js:
node problem1-reverseString.js
See the output in the console.
